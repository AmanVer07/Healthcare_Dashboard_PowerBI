# 🏥 Healthcare Provider Dashboard (Power BI)

## 📌 Overview
This project presents an interactive Power BI dashboard designed to analyze healthcare provider data, focusing on billing, treatment costs, insurance coverage, and patient insights.

## ⚙️ Tools Used
- Power BI
- Excel

## 📊 Features
- KPI tracking (Billing Amount, Medication Cost, Treatment Cost, Insurance Coverage)
- Department-wise analysis (Orthopedics, Neurology, Cardiology, etc.)
- Procedure-wise revenue breakdown (X-Ray, MRI, CT Scan, etc.)
- Diagnosis and service type comparison (Emergency, Inpatient, Outpatient)
- City-wise visualization using map

## 📸 Dashboard Preview
![Dashboard](dashboard.png)

## 🔍 Key Insights
- Orthopedics contributes the highest share (~35%) in total billing
- X-Ray procedures generate the maximum revenue among all procedures
- Emergency services dominate across most diagnoses
- Significant variation observed in billing across departments and procedures

## 🎯 Business Impact
- Helps healthcare providers identify high-revenue departments
- Enables cost optimization by analyzing treatment and medication expenses
- Supports data-driven decision-making for resource allocation

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Use filters (city, department, service type) to explore insights
